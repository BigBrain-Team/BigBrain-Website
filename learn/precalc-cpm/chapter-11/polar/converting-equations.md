# Converting Polar Equations
## Relationships
Remember [lesson 2's Converting Polar Coordinates](chapter-11/polar/converting)?
It's ok if you don't. 

Below is a table for reference:

| $x=r\cos(\theta)$ |     $y=r\sin(\theta)$      |
| :---------------: | :------------------------: |
|   $r^2=x^2+y^2$   | $\tan(\theta)=\frac{y}{x}$ |

## Solving Strategy
1. Know what **type of equation** you are converting too.
2. Replace the appropriate terms.
3. Simplify and **solve!**
	- <u>Complete the square</u> if solving for $x$ and $y$
	- <u>Factor and divide</u> if solving for $r$ and $\theta$
